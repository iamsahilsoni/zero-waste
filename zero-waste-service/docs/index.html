verifyToken

Authenticates user based on the jwt token.

verifyToken
Parameters
req (httpRequest) request from http request, containing token and header to match user credentials
res (httpContent) response sent to route after middleware processing
next (route) to call the next middleware, i.e. to return back control to route which called this middleware for authentication

binSchema

binSchema
Parameters
name (any) Name of the bin provided by the user, e.g. Kitchen bin
pid (any) Physical id of the bin, unique to every bin
userId (any) user of the bin, under whose name is the bin registered
status (any) Full/Fillable - Tells if the bin is full or not

userSchema

userSchema
Parameters
name (any) Full name of user
email (any) Email of the user
password (any) Encrypted password of the user
isAlexaIntegrated (any) if user has integrated Alexa with the system

wasteSchema

wasteSchema
Parameters
binId (any) bin in which the waste is being collected
date (any) date on which the waste is collected
totalWeight (any) total weight collected on a given day
currentWeight (any) current weight of the bin on a given day
